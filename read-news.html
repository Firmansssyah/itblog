Cara Menggunakan “OTC Updater Tool” untuk Mengupdate Windows Phone
Meskipun Microsoft sudah menegaskan bahwa Windows 10 Mobile tidak lagi menjadi fokus utama mereka, tetapi Microsoft menyatakan tetap berkomitmen untuk memberikan support terhadap pengguna Windows Phone — khususnya yang berhubungan dengan security update. Hal itu mereka buktikan — selain dengan merilis patch akan celah keamanan Wi-Fi KRACK lebih cepat dari iOS dan Android — juga dengan dirilisnya tool updater baru untuk Windows Phone agar tetap up-to-date.

Microsoft menamakannya Over-the-Cable (OTC) Updater Tool, yang bisa kamu gunakan untuk smartphone bersistem operasi Windows Phone 8.1 dan Windows 10 Mobile. Tool ini bakal otomatis mendeteksi seri Windows Phone kamu, dan menginstall update paling baru yang tersedia untuk smartphone kamu tersebut.

Bagi kamu yang tertarik untuk menggunakan Over-the-Cable Updater Tool ini, silakan download langsung dari halaman official Microsoft berikut ini:

Ada hal penting yang perlu kamu ketahui seputar OTC Updater tool ini. Untuk menggunakannya Windows Phone / Windows 10 Mobile kamu harus versi final (non-insider), jadi jika masih berada di program insider, silakan keluar dari program Windows Insider melalui Settings > Update & Security > Windows Insider Programme > Stop Insider Preview Build.

Jika build Windows 10 Mobile yang kamu gunakan adalah build Insider, maka kamu harus reset dulu menggunakan WDRT. Intinya, Windows Phone 8.1 / Windows 10 Mobile yang kamu gunakan haruslah versi public non-insider. (Baca: Cara Recovery Lumia dengan WDRT )

Windows Phone kamu juga harus dalam kondisi airplane mode, serta baterai dalam kondisi lebih dari 40%. Oh ya, PC kamu juga harus sudah terinstall .NET 4.0 ya, jika belum kamu bisa download disini.
